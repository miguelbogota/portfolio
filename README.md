# Miguel Bogota Portfolio

Personal portfolio using Angular 10 connected to firebase as a database. It works as a PWA and it'll list all the different projects I've been in. 

## [Live Website](http://miguelbogota.tk/)

## Features

* [Angular 10](https://angular.io/) - Front end framework used
* [Firebase](https://firebase.google.com/) - Back end used
